## 🎯 Objetivo

Construir una solución analítica que permita entender la operación comercial de HelixCore, enfocándose en:

1. **Funnel comercial** — Cuellos de botella, conversiones y diferencias entre segmentos.
2. **Contactabilidad** — Gestión del contacto a leads, fugas e ineficiencias.
3. **Performance comercial** — Señales sobre reps, descuentos, bots e iniciativas.

## 📊 Datos

Los datasets se encuentran en la carpeta `data/`. Consulta `data/data_dictionary.csv` para la descripción de cada campo.

| Archivo | Descripción |
|---|---|
| `leads.csv` | Leads generados con información de segmento, país y canal |
| `deals.csv` | Oportunidades/deals del pipeline comercial |
| `funnel_stage_history.csv` | Historial de cambios de etapa en el funnel |
| `sales_activities.csv` | Actividades de ventas (llamadas, emails, etc.) |
| `call_logs.csv` | Detalle de llamadas realizadas |
| `bot_interactions.csv` | Interacciones con bots de IA |
| `product_signals.csv` | Señales de producto (PQL score, onboarding, etc.) |
| `rep_daily_capacity.csv` | Capacidad diaria de los reps de ventas |
| `reps.csv` | Información de los representantes de ventas |
| `subscriptions.csv` | Datos de suscripciones |
| `partner_portfolios.csv` | Portafolios de partners contables |
| `data_dictionary.csv` | Diccionario de datos |
